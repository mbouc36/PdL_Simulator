# PdL_Simulator

Contains code for IMU (MinIMU-9) and ToF (VL53L0X) sensors as well as Load Cell Amp (HX711) interfacing with Aruino Uno

<img width="476" height="541" alt="image" src="https://github.com/user-attachments/assets/9358fd2e-0a77-4c9c-b920-438085ae9098" />


## Install Requirements
`pip install pip-tools`
`pip install -r requirements.txt`

## Setup Config
tip: To find serial ports on Mac run: `ls /dev/cu.*`
`python3 update_config.py`
